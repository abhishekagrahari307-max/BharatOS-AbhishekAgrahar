# भारत ओएस - प्रारंभिक मार्गदर्शिका

## प्रणाली आवश्यकताएँ
- 64-बिट प्रोसेसर
- 4GB RAM (8GB अनुशंसित)
- 20GB डिस्क स्थान
- TPM 2.0 चिप

## स्थापना चरण
1. डाउनलोड ISO फ़ाइल
2. USB ड्राइव पर बूट करने योग्य बनाएं
3. भारत ओएस बूट करें
4. योग प्रमाणीकरण पूरा करें

- शून्य-विश्वास सुरक्षा
